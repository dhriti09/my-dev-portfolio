
# 🌐 Dhriti Yadav | Portfolio

A sleek, responsive portfolio website built to showcase my technical skills and academic journey as a **B.Tech CSE student**. This site features a glassmorphism design and interactive elements to highlight my work in **Full-Stack Development** and **Competitive Programming**.

## 🚀 Features

* **Responsive Design**: Optimized for all devices using Bootstrap’s grid system.
* **Glassmorphism UI**: Modern aesthetic with frosted-glass effects and smooth transitions.
* 
**Skill Progress Tracking**: Dynamic animation of technical proficiencies like C++, JavaScript, and Python.


* **Interactive Project Gallery**: Direct links to my GitHub repositories for real-time code review.
* **Performance Focused**: Lightweight JavaScript for smooth scrolling and intersection-based reveal animations.

## 🛠️ Technologies Used

| Technology | Purpose |
| --- | --- |
| **HTML5** | Core structure and semantic content. |
| **CSS3** | Custom styling, glassmorphism effects, and responsive layouts. |
| **JavaScript (ES6+)** | DOM manipulation, scroll animations, and form validation. |
| **Bootstrap 5** | Frontend framework for rapid, mobile-first development. |
| **Intersection Observer API** | Powering the "Reveal on Scroll" animations. |
